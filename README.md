# Interfacing with the Panasonic AMG8833 Evaluation Kit


## Required Libraries (included in data folder)
1. Wire.h
2. Grideye.h

## Circuit Connections
Arduino Pin | Sensor board Pin
------------|-----------------
A4(SCL)     |       D14
A5(SDA)     |       D15
5V          |       5V
GND         |       GND
